  2. Misinformation Game
    • Link: Misinformation Game
    • Pre-Reqs: None
    • Suggested Length: 30 min
    • Tech Dependencies: Google Chrome
    • Key Concepts: Spread of Misinformation, Fake News
    • Group Reqs: The whole class will play together (or split into smaller groups).
    • Description: Students encounter several news articles, half of which have key features of misinformation and the other half does not. Students engage in a mock information spread activity where they make decisions about which news items they encounter should be published, and which ones should be discarded. We visualize which items were spread more. We discuss the six key features of misinformation and how it is shown to spread more than authentic information.
